# 📚 Wetoro Setup Guides for Non-Coders

Welcome! This folder contains **simple, step-by-step instructions** to help you set up Wetoro, even if you've never coded before.

## 🎯 What You Need to Know

Wetoro is a web application that lets people release emotional stones into a collective clearing. To make it work, you need to:
1. Set up a place to store the stones (database)
2. Set up a way for the website to talk to the database (API server)
3. Put your website online so people can use it

Don't worry - we'll walk you through everything!
